# Two-Factor Authentication (2FA)

As a security best practice, Robins & Morton uses multi-factor authentication (MFA) or two-factor authentication (2FA) for access to Egnyte and Office365, including Outlook and other applications, HUB access, and SharePoint. For more information, please reference the following articles:
