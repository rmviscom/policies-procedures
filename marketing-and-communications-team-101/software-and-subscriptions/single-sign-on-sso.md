# Single Sign-on (SSO)

Single sign-on (SSO) is an authentication method that allows a user to log in with a single user ID and password to any of several related, yet independent, software systems.

With SSO, you can use your Robins & Morton username and password to log in to many company apps including:

* Microsoft
* Egnyte
* Unanet
* OpenAsset
* HUB
