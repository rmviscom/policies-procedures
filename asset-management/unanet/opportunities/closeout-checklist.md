## Checklist by app

### Egnyte

* [ ] Ensure RFP and Presentation files are saved in [Marketing Opportunities](https://robinsmorton.egnyte.com/navigate/folder/4e33d5bb-9cbc-45de-a68a-c2884a3e6cf8) Egnyte Folder.
* [ ] Organize/clean up working files and duplicates.
* [ ] Ensure folder names match with Unanet. Include Opportunity Number at the end of Opportunity name. _(Ex. Pineville AHP Cath Lab 22-0461)_ This will help team members find pursuits.
* [ ] Save strategic pursuit workbook (if used) in [Sales Strategy](https://robinsmorton.egnyte.com/navigate/folder/49d28f69-e42e-4d40-8ec9-f331d6a6fae0) folder.



### Unanet

* [ ] Ensure the opportunity has been updated in Unanet. Continually add any additional information throughout the life of the opportunity.
* [ ] General Info: Name, Stage, Secondary Categories, etc.
* [ ] Metrics: Fee %, Competition, Insurance, General Conditions, etc.
* [ ] Lifecycle: RFP/Presentation Dates, Qualitative Score.
* [ ] Documents: Upload Client RFP Files, RFP Response, Presentation Documents (PDF, word, PowerPoint)
* [ ] Personnel Updates: Record any personnel updates not currently in Unanet (i.e. years in the industry, project discrepancies, references or any other customized/detailed information collected).
* [ ] Add Estimated Start Date, Estimated Completion Date and Duration (pushed to Bridgit).
* [ ] Project Updates: Record any new information not currently in the project profile in Unanet.
* [ ] Client references: Record any new references: (Owner/Client and Architect).
* [ ] Update competition (short listed firms) and who was awarded the project.
* [ ] Save debrief notes in time stamped notes.
* [ ] Add project information to Projects - change order percentage, references, metrics, etc.



### OpenAsset

* [ ] Save any new items to OpenAsset to the necessary category including:
  * [ ] renderings
  * [ ] CM Builder renderings or videos
  * [ ] new project or stock photos **(project photos must be reviewed by safety)**
  * [ ] client/agency/trade logos, etc.&#x20;



### Joist

* [ ] Work with Serena to add new boilerplate responses in Joist.
* [ ] Upload final RFP response in Joist.



### CC Library

* [ ] Flag any graphics or content blocks that would be useful for other marketers.
* [ ] Alert the graphics team to have them added to the CC Library.



### Monday.com

* [ ] Ensure the opportunity was added to Monday.com [Sales Pursuits board](https://robinsmorton-global.monday.com/boards/4259861171) and all fields are completed.
* [ ] Include Opportunity Number.
* [ ] Include marketing hours for the pursuit: pre-proposal, proposal and interview.



### Teams

* [ ] Share any new content/graphics with Marketing on Teams. All RFPs can be accessed in the Presentations/Proposals YTD report in Monday.com.
