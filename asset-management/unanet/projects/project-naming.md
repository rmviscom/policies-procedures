# Project Naming



<figure><img src="../../../.gitbook/assets/UnanetProjectNames (1).png" alt=""><figcaption><p>Example: Project Name and Other Project Name</p></figcaption></figure>

## Project Name = _Facility Name with description identifier_

_Example:_ Joe DiMaggio Children's Hospital Vertical Expansion&#x20;

## Other Project Name **=** _Facility Name_&#x20;

_Example:_ Joe DiMaggio Children's Hospital&#x20;

## Standard Description = _New Basement Entrance_

{% hint style="info" %}
When entering the Project Name or the Other Project Name, be sure to use the specific facility name, not the owner name.&#x20;

_For example:_

Duke Health is the owner. The facility name would Duke University Medical Center or Duke Raleigh Hospital. **Use Google and/or Definitive Healthcare to get formal facility name.**
{% endhint %}
