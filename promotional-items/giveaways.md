## Stocked tradeshow giveaways

[In-stock giveaway items for tradeshows or other events can be found on the Company Store.](https://rmcs.conceptsusa.com/products?s%5Bf%5D%5Bc%5D%5B%5D=%2FPromotional)

## Custom giveaway items

If you would like to have a custom event item created for the tradeshow, email [Julianna Danner](mailto:jdanner@conceptsusa.com) or use the [Special Requests tab on the company store](https://rmcs.conceptsusa.com/pages/35816) to start this process. When making a special request, please include:&#x20;

* the type of event, item, or type of item desired
* estimated budget
* quantity
* the event date

If you have any questions or need any assistance while sourcing or creating promotional items for events, please contact [Amanda Bradley Witteck](mailto:abradley@robinsmorton.com).
