# OS-005 Project Start-Up Activities List

1\. The jobsite Project Start-Up Activities List (form OS-145) will be initiated at project award, updated weekly, and will be used to verify items to be accomplished and confirm their completion.

2\. The project specification and local conditions will be reviewed to determine any special start-up requirements. Any items found will be added to the Project Start-Up Activities List.

3\. This list will be reviewed in the Project “Kick off” Meeting as outlined in Policy OS-050
